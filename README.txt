ZyroDevv Cloud GitHub Preview Static

Ini hanya untuk preview tampilan di GitHub Pages.
Tidak ada PTLA, PTLC, FR3 API key, bot token, atau secret lain di sini.

Cara pakai:
1. Extract ZIP.
2. Upload index.html dan CNAME ke root repo GitHub.
3. Settings > Pages > Branch main / root.
4. Custom domain: zyrodevv.web.id.

Catatan:
- Tombol checkout hanya simulasi.
- Payment FR3 dan auto create panel wajib pakai backend Node.js private di hosting/panel.
