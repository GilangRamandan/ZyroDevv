Menjalankan Bot NodeJS

  1. Pilih jenis server NodeJS semasa melakukan pembelian.
  2. Buka URL Panel Pterodactyl dan masuk dengan butiran yang diberikan.
  3. Muat naik folder fail projek NodeJS anda ke File Manager.
  4. Tetapkan fail permulaian (startup file) seperti index.js.
  5. Klik Start untuk memulaikan bot anda!

Mengaktifkan Server Minecraft

  1. Pilih jenis server Minecraft and jenis egg yang sesuai.
  2. Log masuk ke panel, setuju dengan EULA di dalam fail eula.txt.
  3. Tetapkan versi Minecraft melalui menu permulaian panel.
  4. Mulai semulai server dan berikan IP serta Port kepada pemain anda.
