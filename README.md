# ZyroDevv Cloud Static Demo

Website static 1 file untuk GitHub Pages.

## Cara upload GitHub Pages
1. Upload `index.html` ke repository GitHub.
2. Aktifkan GitHub Pages dari Settings > Pages.
3. Buka link GitHub Pages.
